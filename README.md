# CSM
Canvas Self Montage


Esta aplicación convierte archivos PDF y Word a HTML, adaptados para Canvas específicamente.

## Requisitos

- Python 3.6+
- PyMuPDF
- pdfplumber
- PyPDF2
- python-docx
- mammoth
- PyQt5

Proyecto en Pausa.

## Instalación

```bash
pip install -r requirements.txt
